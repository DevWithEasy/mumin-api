# express-mern-x

Create a expres server for mern stack.
it will be helpfull to easy setup a express server.Lets try and enjoy
.
## Create a project

```javascript
npx express-mern-x project_name
or
npx express-mern-x
```

## Run development

```javascript
npm run dev
```

## Users package

| Use             | Package                                                                |
| ----------------- | ------------------------------------------------------------------ |
| Server | express |
| Database | Database |
| Authentication | jsonwebtoken |
| Hashing | bcrypt |
| File Upload | multer |
